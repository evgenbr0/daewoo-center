# Daewoo Center — сайт автосервиса (daewoocenter.ru)

Готовый статический сайт независимого специализированного автосервиса по обслуживанию
и ремонту автомобилей Daewoo. Сделан по техническому заданию: чистый HTML/CSS/JS, без
бэкенда, базы данных и сборщиков. Полностью адаптивный, работает сразу после копирования
на хостинг.

---

## 📁 Структура проекта

```
daewoocenter.ru/
├── index.html              # Главная (и единственная) страница сайта
├── assets/
│   ├── css/
│   │   └── style.css       # Все стили
│   ├── js/
│   │   └── main.js         # Интерактив: меню, форма, FAQ, прайс-аккордеон, анимации
│   └── img/
│       └── favicon.svg     # Иконка сайта (зелёная шестерня с буквой D)
├── .nojekyll               # Отключает обработку Jekyll на GitHub Pages
└── README.md               # Этот файл
```

## 🚀 Как разместить на хостинге (инструкция)

Скопируйте всю папку `daewoocenter.ru` (файл `index.html` и папку `assets`) в корневую
директорию вашего хостинга — обычно это `public_html`, `www` или `htdocs`. Файл
`index.html` должен лежать в корне, а папка `assets` — рядом с ним. После загрузки сайт
сразу откроется по адресу домена без какой-либо настройки сервера, сборки или установки
зависимостей. Никаких баз данных и серверных языков не требуется.

> Локально сайт также можно открыть, просто дважды кликнув по `index.html` в браузере.

## ✨ Что есть на сайте

- **Главный экран** с чётким позиционированием (профильный сервис Daewoo), геометрическим
  фоном-сеткой и зелёной карточкой онлайн-записи.
- **Услуги** — 10 услуг в виде карточек с крупными номерами и зелёными иконками.
- **Преимущества** — чередующиеся сплит-строки (иконка/текст поочерёдно слева–справа).
- **Модели Daewoo**, которые обслуживаются (Nexia, Matiz, Lanos, Gentra, Nubira, Espero,
  Leganza, Tacuma, Kalos, Magnus).
- **Цены** — прайс-аккордеон по категориям (ТО, двигатель, подвеска, тормоза, электрика,
  кузов); открыта всегда одна группа.
- **Этапы работы** — вертикальный таймлайн из 6 шагов.
- **Отзывы** клиентов — сетка карточек.
- **Форма записи** (имя, телефон, модель, услуга) с валидацией и сообщением об успешной отправке.
- **FAQ** — аккордеон с частыми вопросами.
- **Контакты** со схемой проезда (плейсхолдер) и всеми данными.
- **Адаптив**, мобильное меню-бургер, плавающая панель действий (Позвонить / Записаться),
  кнопка «наверх».

Все контактные данные (телефон, адрес, e-mail, мессенджеры, реквизиты) намеренно
заменены заглушками **ХХХХХ** согласно ТЗ — их нужно заменить на реальные перед публикацией.

---

## 🔧 Что заменить перед запуском

1. Контактные заглушки `ХХХХХ` → реальные телефон, адрес, e-mail, мессенджеры, реквизиты
   (в `index.html`, включая блок Schema.org в `<head>` и значения `tel:`/`mailto:`).
2. При необходимости — реальную карту (Яндекс.Карты / Google Maps) вместо плейсхолдера.
3. Цены в прайсе — на актуальные (сейчас указаны демонстрационные значения).
4. Форма записи имитирует отправку на стороне браузера. Чтобы заявки приходили на почту/в
   CRM, подключите сторонний обработчик форм (например, Formspree).

## 🛠 Технологии

- Семантический **HTML5**, разметка **Schema.org** (AutoRepair) для SEO.
- **CSS3**: кастомные свойства, Grid/Flexbox, сплит-строки, прайс-аккордеон, адаптивные
  сетки, анимации, `prefers-reduced-motion`.
- **Ванильный JavaScript** (без библиотек): прайс-аккордеон «открыт только один», валидация
  форм с доступностью (aria-invalid, role=alert), маска телефона, мобильное меню с управлением
  фокусом (focus-trap, inert), аккордеон FAQ, анимации появления по скроллу, кнопка «наверх».
- Шрифты **Rubik** (заголовки) и **Inter** (текст) — оба с кириллицей — подключены через
  Google Fonts (CDN). Есть системные фолбэки — сайт остаётся читаемым даже без доступа к CDN.

Внешних зависимостей, требующих сборки или установки, **нет**.
